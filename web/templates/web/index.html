Proyecto Matematico De Apoyo Estudiantil

Intuitivo

La platorma desarrollada a traves del proyecto matematico de apoyo estudiantil permite a los estudiantes de secundaria reforzar el conocimiento adquirido en las aulas de manera facil e intuitiva

Seguridad

Implementamos un uso de usuarios con un registro preehecho para tener la fácilidad y rápidez de conectarse.

Accesibilidad

Tratamos de introducir a los usuarios en una página adaptativa para entender su uso de la forma más sencilla.

Sobre el proyecto

  • ¿Cuando?

    Este proyecto inicia en el año 2022, presentando un apoyo hacia todos los estudiantes de secundaria para darles capacitación durante sus cursos de matemáticas.

  • ¿Porque?

    Estamos concientes de que al ser una materia muy pesada o aburrida para ciertos jovenes, pueden llegar a odiar la materia o tener un pesimo rendimiento, por ello nació esta ídea, para darle una estructura más dinámica a las matemáticas.

  • ¿Como?

    Otorgando una web personal y privada para el desarrollo lógico matemático propio de cada grupo así como de cada estudiante.

  • Resultado

    Como resultado tenemos este sitio para asesoria de temas y prácticas para que en cualquier momento el estudiante pueda entrar y revisar cada tema
